# Setup

## Nodeバージョン
Nodeバージョンは `.nvmrc` 記載のバージョンに指定してください。  
`nvm`を使用している場合は、`nvm use`で指定バージョンに切り替えができます。

## ローカルサーバー起動
```bash
npm install
npm run dev
```

## generate
```bash
npm run generate
```

## Locally preview production build:
```bash
npm run preview
```
